# 2D Space Shooter - Arcade Game Development

A Galaxian-inspired 2D arcade game developed using **Java** and the **Swing/AWT** libraries. This project was created as a formal assignment for the **Object-Oriented Programming (OOP)** course at the University of Riau.

## Project Overview
This project demonstrates the implementation of core game mechanics and software architecture principles. Although initiated as a **Group Project**, I served as the **Lead Developer**, responsible for **98% of the total development**, including system logic, physics, and asset integration.

### Key Features:
* **Dynamic Enemy Waves:** Intelligent spawning system with varying patterns.
* **Real-time Collision Detection:** Precise interaction between projectiles and entities.
* **Responsive Controls:** Smooth player movement and shooting mechanics.
* **Asset Integration:** Seamless implementation of external visual and audio resources.

## Technical Implementation & Learning
* **OOP Principles:** Applied inheritance, encapsulation, and polymorphism to manage game states and entities efficiently.
* **AI-Assisted Development:** Leveraged AI tools for debugging and optimizing complex algorithms, significantly accelerating the development lifecycle and improving code quality.
* **Resource Management:** Managed external assets while ensuring optimal performance and memory usage.
